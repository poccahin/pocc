# Tensor Telepathy Engine

`tensor_telepathy_engine` defines a collaboration channel where agents exchange high-dimensional intent vectors instead of natural language.

## Design focus
- Intent projection into a shared latent space.
- Cosine resonance filtering for candidate collaborators.
- Conflict-aware batching for large swarm coordination rounds.

## Status
This module is currently a scaffold and will be expanded with Rust/Python interop prototypes.
